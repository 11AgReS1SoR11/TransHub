<br/>Данный пункт меню позволяет открыть окно с информацией о состоянии всех загруженных компонентов комплекса.
<br/>Для открытия данного окна зайдите в раздел меню "О системе" -> "Статусы компонентов".
<br/>
<br/><center><img src="menu_about_system.png" /><br/><i>Меню</i></center>
<br/><center><img src="components_state.png" /><br/><i>Окно информации о статусах компонентов</i></center>
<br/>
<br/>Левый край окна представляет собой дерево компонентов.
<br/>Данное дерево может содержать несколько корневых узлов:
<br/>1) Запущенные - содержит список запущенных компонентов.
<br/>2) Не запущенные - содержит список не запущенных компонентов.
<br/>3) Не загруженные - содержит список не загруженных компонентов (файлов).
<br/>
<br/><font color=red><b>Корневые узлы "Не запущенные" и "Не загруженные" могут отсутствовать, если запуск всех компонентов был успешен!</b></font>
<br/>
<br/>При выборе конкретного компонента в дереве будет выведена вся доступная по нему информация.
