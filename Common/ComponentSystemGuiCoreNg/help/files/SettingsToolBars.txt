<br/>Панели инструментов служат для быстрого доступа к часто используемым функциям.
<br/>Для настройки панелей инструментов зайдите в раздел меню "Инструменты" -> "Параметры..." и в появившемся окне выберите пункт "Панели инструментов".
<br/>
<br/><center><img src="menu_settings.png" /><br/><i>Меню</i></center>
<br/><center><img src="settings_toolbars.png" /><br/><i>Окно параметров</i></center>
<br/>
<br/>Данный раздел настроек позволяет показать/скрыть необходимые панели инструментов, а так же зафиксировать их положение.
<br/>
<br/><b>Настройка видимых панелей инструментов:</b>
<br/>1) Для настройки видимых панелей инструментов включите те панели, которые следует отображать.
<br/>2) Нажмите кнопку "Применить".
<br/>
<br/><b>Настройка перетаскивания:</b>
<br/>1) Для разрешения перетаскивания, включите "Разрешить перетаскивание" у необходимой панели инструментов.
<br/>2) Нажмите кнопку "Применить".
<br/>
<br/>Все произведенные настройки автоматически сохраняются после завершения работы комплекса.
