<br/>Состояние уведомлений комплекса и управление ими осуществляется при помощи иконки в строке состояния.
<br/>При наведении курсора мыши на иконку уведомлений выводится всплывающее окно с информацией:
<br/>1) Активные уведомления - количество уведомлений, не обработанных пользователем.
<br/>2) Всего уведомлений - суммарное количество уведомлений. 
<br/>
<br/><center><img src="main_window_popup_0.png" /><br/><i>Уведомлений нет</i></center>
<br/><center><img src="main_window_popup_1.png" /><br/><i>Уведомления есть</i></center>
<br/>
<br/>Для просмотра списка уведомлений следует нажать на иконке левой кнопкой мыши.
<br/>В появившемся окне будет содержаться список уведомлений с полями:
<br/>1) Тип уведомления
<br/>2) Время добавления
<br/>3) Название
<br/>4) Описание
<br/>
<br/><center><img src="main_window_popup_list.png" /><br/><i>Окно уведомлений</i></center>
<br/>
<br/>Так же в данном окне имеется контекстное меню, вызываемое по нажатию правой кнопки мыши.
<br/>Меню содержит:
<br/>1) Удалить из списка - удалить уведомление из списка.
<br/>2) Очистить список - очистить весь список уведомлений.
<br/>3) Проматывать в конец - при добавлении нового уведомления и открытом окне, список уведомлений будет проматываться в конец к новому добавленному уведомлению.
<br/>
<br/><center><img src="main_window_popup_list_context_menu.png" /><br/><i>Контекстное меню окна уведомлений</i></center>
<br/>
<br/><b>Типы уведомлений:</b>
<br/>1) Информация
<br/>2) Норма (или ОК)
<br/>3) Предупреждение
<br/>4) Критическое
<br/>

<br/><center><img src="main_window_popup_warning.png" /><br/><i>Уведомление "Предупреждение"</i></center>
<br/><center><img src="main_window_popup_err.png" /><br/><i>Уведомление "Критическое"</i></center>
<br/>
<br/><b>Закрытие уведомлений:</b>
<br/>1) Уведомления "Информация" и "Норма (или ОК)" закрываются автоматически через 3 сек с момента открытия.
<br/>2) Уведомления "Предупреждение" и "Критическое" закрываются только при нажатии кнопки "Закрыть" на уведомлении.
