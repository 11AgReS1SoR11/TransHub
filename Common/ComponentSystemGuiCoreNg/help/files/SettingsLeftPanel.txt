<br/>Левая информационная панель служит для компактного вывода необходимой информации.
<br/>Для настройки левой информационной панели зайдите в раздел меню "Инструменты" -> "Параметры..." и в появившемся окне выберите пункт "Левая информационная панель".
<br/>
<br/><center><img src="menu_settings.png" /><br/><i>Меню</i></center>
<br/><center><img src="settings_left_panel.png" /><br/><i>Окно параметров</i></center>
<br/>
<br/>Для настройки доступны следующие параметры:
<br/>1) Показывать панель - показывать или скрывать информационную панель.
<br/>2) Фиксировать размер - зафиксировать ширину информационной панели.
<br/>3) Автоматически скрывать - автоматически скрывать информационную панель, если курсор мыши находится не над ней.
<br/>4) Перетаскивание вкладок - разрешает/запрещает перетаскивание вкладок на информационной панели.
<br/>5) Расположение вкладок - позволяет настроить расположение вкладок на информационной панели.
<br/>6) Вкладки - позволяет показать/скрыть отдельные вкладки.
<br/>
<br/><b>Настройка ширины панели:</b>
<br/>1) Для настройки необходимой ширины панели отключите фиксацию размера панели и нажмите кнопку "Применить".
<br/>2) Поднесите курсор мыши к краю панели пока курсор не изменит свой вид на двустороннюю стрелку и с нажатой левой клавишей мыши задайте новую ширину панели.
<br/>3) Включите фиксацию размера панели и нажмите кнопку "Применить".
<br/>
<br/><center><img src="left_panel_size.png" /><br/><i>Изменение размера левой информационной панели</i></center>
<br/>
<br/><b>Настройка автоматического скрытия панели:</b>
<br/>1) Для автоматического скрытия панели включите чекбокс "Автоматически скрывать" и нажмите кнопку "Применить".
<br/>2) Для отключения автоматического скрытия панели отключите чекбокс "Автоматически скрывать" и нажмите кнопку "Применить".
<br/>Панель будет автоматически скрываться, если курсор мыши покидает ее область, и будет автоматически появляться при поднесении курсора мыши к левому краю экрана.
<br/>
<br/><center><img src="left_panel_show.png" /><br/><i>Положение курсора мыши для автоматического появления левой информационной панели</i></center>
<br/>
<br/><b>Настройка перетаскивания вкладок:</b>
<br/>1) Для перетаскивания вкладок включите чекбокс "Перетаскивание вкладок" и нажмите кнопку "Применить".
<br/>2) Для отключения перетаскивания вкладок отключите чекбокс "Перетаскивание вкладок" и нажмите кнопку "Применить".
<br/>
<br/><center><img src="left_panel_move_tabs.png" /><br/><i>Перетаскивание вкладок</i></center>
<br/>
<br/><b>Настройка расположения вкладок:</b>
<br/>1) Для настройки расположения вкладок на левой информационной панели выберите соответствующий пункт комбобокса и нажмите кнопку "Применить".
<br/>2) Для настройки расположения вкладок по умолчанию (для левой панели - по левому краю) выберите соответствующий пункт комбобокса и нажмите кнопку "Применить".
<br/>
<br/><center><img src="base_panel_tabs_pos.png" /><br/><i>Расположение вкладок</i></center>
<br/>
<br/><b>Настройка видимых вкладок:</b>
<br/>1) Для настройки видимых вкладок включите чекбокс "Вкладки" и отключите вкладки, которые не следует отображать.
<br/>2) Нажмите кнопку "Применить".
<br/>
<br/>Все произведенные настройки автоматически сохраняются после завершения работы комплекса.
<br/>Так же сохраняется порядок вкладок и последняя активная вкладка.
