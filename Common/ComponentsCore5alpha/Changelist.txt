** 17.05.2021 - Кромачев Максим - Версия 1.2.0

-- Удалён метод разбора входных параметров (parseOptions). 
   Разбор входных аргументов перенесён из библиотеки в приложение. Вместо метода parseOptions добавлены два отдельных метода:
   static void addArguments (const QMap<QString, QString> &arguments) - добавляющий аргументы всем компонентам, у которых в описании есть данный аргумент;
   static void addArguments (const QString &component, const QMap<QString, QString> &arguments) - добавляющий аргументы указанному компоненту, если у компонента в описании он есть.
   ВАЖНО - методы не проверяют наличие уже добавленных аргументов, и просто перезатирают уже существующие аргументы!
   
-- Добавлена возможность отключать/включать отдельные компоненты (в методе setComponentPaths).

** 19.02.2021 -  - Версия 1.1.2

-- Исправлена ошибка метода isInitDone. 
-- Удалены сигналы signalStartSuccess, signalStartFailed. Введён сигнал signalInitDone.

   очереди загрузки/инициализации, будет выслан сигнал signalInitDone.
